# 世纪星空调通信协议逆向

逆向世纪星中央空调温控面板通信协议，目标：保留原装面板 + ESP32旁路接入智能家居。

## 项目状态

**当前阶段**：协议解码中 —— 已捕获A/B线脉冲宽度编码信号，确认三态（0/1/E）7-bit帧结构。

## 硬件

| 设备 | 说明 |
|------|------|
| 世纪星温控面板 | 墙上原位，仅A/B两线出线，非强电 |
| 世纪星黑盒子 | 吊顶内，6端子(低/中/高/零/火/信号线)，信号线仅连风机 |
| ESP32 DevKit V1 | MicroPython v1.24.1，**仅用D33**（D14已烧毁，D18已衰减） |
| 104电容 | 100nF/400V，信号耦合用，脚易断需备货 |

**当前接线**：B线 → 104电容 → ESP32 D33，A线 → ESP32 GND（直连）

## 协议摘要

- **物理层**：单线脉冲宽度编码，~485μs=0，~975μs=1，~1250-1600μs=E（第三态/行扫描标记），>4000μs=F（帧心跳分隔符）
- **帧结构**：7-bit为主，F脉冲间隔~395ms，按键时帧密度从2.4fps暴增至~50fps
- **编码模型**：面板持续广播状态寄存器值，非"按一次发一个命令"
