## 拓扑事实（已确认，不可推翻）

1. **面板**：仅有A/B两根出线，非强电
2. **黑盒子**：6根端子线 = 低(Lo) + 中(Md) + 高(Hi) + 零(N) + 火(L) + 信号线(S)
3. **信号线(S)**：仅连接吊顶风机，不连任何其他地方
4. **A/B线**：入墙后与黑盒子线缆在墙内纠缠/转换，具体方式不可知（需拆墙）
5. **A/B线与黑盒子信号线(S)绝对不相连**
6. **命名规则**：仅用A线/B线，**严禁颜色描述**
7. **ESP32**：只能接低压信号线，禁止接220V AC

## 接线历史

### 当前有效接线 ✅（2026-07-05起）

```
ESP32 D33 ← B线（面板出线，直连，无104电容）
ESP32 GND ← A线（面板出线，直连）
ESP32 USB → 电脑 USB
```

**B线电压测量**（07-05）：
- 空闲电压：2.32V
- ESP32 拉低后：0.43V（Δ=1.89V）
- 物理层回放可行

**关键发现**：104电容已拆除。B线实测 2.32V，ESP32 3.3V 输出可直接拉低。

### 待实施接线（继电器隔离方案）

**目的**：回放验证时让面板脱开总线，ESP32 独占B线。

```
5V继电器模块:
  DC+   → ESP32 5V
  DC-   → ESP32 GND
  IN    → ESP32 D25（控继电器吸合）
  跳线  → Low 触发（D25=1 时吸合）
  
  COM   → 主机B线（剪断后接主机那一端）
  NC    → 面板B线（剪断后接面板那一端）
  NO    → ESP32 D33
  
A线保持原样（ESP32 GND 接 A线）

平时：D25=0 → 继电器 NC 闭合 → 面板正常
回放：D25=1 → 继电器吸合 → COM-NO 接通 → ESP32 独占B线
```

### 已废弃接线

| 接线 | 时间 | 废弃原因 |
|------|------|----------|
| GPIO14←黑盒子端子, GND←黑盒子端子 | 06-18 | 仅捕获~92Hz心跳，非数据线 |
| GPIO14←A线(串104电容), GND←B线 | 06-20~27 | D14被100kΩ上拉烧毁 |
| D18←A线(串104电容), GND←B线 | 06-27 | D18信号时断时续，接触不良 |
